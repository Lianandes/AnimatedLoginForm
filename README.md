# Animated Login Form

Projeto desenvolvido para praticar animações e estilização de formulários utilizando HTML e CSS através de uma interface moderna de login.

## 🚀 Objetivo

O objetivo deste projeto foi aprimorar conhecimentos em desenvolvimento front-end, animações CSS e construção de formulários modernos e interativos.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Formulário de login animado
- Efeitos visuais modernos
- Transições suaves
- Layout responsivo
- Interface organizada

## 📚 Conceitos praticados

- CSS Animations
- Transitions
- Estilização de formulários
- Responsividade
- Organização visual
